# EKWA Before After Gallery

A beautiful, modern, and highly customizable before-and-after gallery plugin for WordPress. Perfect for dental, medical, beauty, fitness, and any business that wants to showcase transformation results.

## Features

- **Beautiful Stacked Card Design**: Modern, elegant card layout with vertically stacked before/after images.
- **Multi-Level Filtering**: Filter by main categories and sub-categories for easy navigation.
- **Multiple Image Sets per Case**: Add multiple before/after angles for each case to showcase comprehensive results.
- **Responsive Design**: Fully responsive and mobile-friendly, looks great on all devices.
- **Modal Gallery**: Full-featured lightbox with navigation, zoom, and swipe support.
- **Easy Management**: Simple and intuitive admin interface for managing cases and image sets.
- **Customizable Appearance**: Change accent colors, titles, and more from the settings panel.
- **Shortcode Support**: Easily embed galleries anywhere with flexible shortcode parameters.
- **Category & Subcategory Filters**: Organize and filter cases for better user experience.
- **SEO Friendly**: Optimized markup for better search engine visibility.
- **Translation Ready**: Includes .pot file for easy translation.
- **Lightweight & Fast**: Minimal impact on site performance.
- **Watermarking & Removal**: Add or remove watermarks on images for branding or privacy.
- **Flexible Image Upload**: Add before and after images as a single combined image or separately.
- **Gallery Settings**: Customize card design and set cards per row for layout flexibility.

## Shortcode Usage

Basic usage:

```
[ekwa_gallery]
```

With parameters:

```
[ekwa_gallery title="Our Results" subtitle="See the difference" category="cosmetic" limit="6" show_filter="yes"]
```

### Parameters

- `title` - Gallery title (default: "Smile Transformations")
- `subtitle` - Gallery subtitle
- `category` - Filter by category slug
- `limit` - Maximum number of cases to show (-1 for all)
- `show_filter` - Show/hide category filters ("yes" or "no")

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'BA Gallery' in the admin menu to add cases
4. Add the shortcode `[ekwa_gallery]` to any page or post

## Gallery Settings

- Card Design: Choose from multiple card styles
- Cards Per Row: Set the number of cards displayed per row

## Flexible Image Upload
