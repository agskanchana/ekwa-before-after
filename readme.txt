A beautiful before and after gallery with stacked card design for dental and medical practices.

== Description ==

EKWA Before After Gallery is a powerful and elegant plugin for showcasing transformations, results, and case studies. Perfect for dental practices, medical offices, beauty salons, fitness centers, and any business that wants to display before and after results.

= Features =

* **Beautiful Stacked Card Design** - Modern and elegant card layout with vertically stacked before/after images
* **Multi-Level Filtering** - Filter by main categories and sub-categories
* **Multiple Image Sets** - Add multiple before/after angles for each case
* **Responsive Design** - Looks great on all devices
* **Modal Gallery** - Full-featured lightbox with navigation
* **Easy Management** - Simple admin interface for managing cases
* **Customizable** - Change colors, titles, and more
* **Shortcode Support** - Easy embedding with flexible parameters

= Shortcode Usage =

Basic usage:
`[ekwa_gallery]`

With parameters:
`[ekwa_gallery title="Our Results" subtitle="See the difference" category="cosmetic" limit="6" show_filter="yes"]`

= Parameters =

* `title` - Gallery title (default: "Smile Transformations")
* `subtitle` - Gallery subtitle
* `category` - Filter by category slug
* `limit` - Maximum number of cases to show (-1 for all)
* `show_filter` - Show/hide category filters ("yes" or "no")

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'BA Gallery' in the admin menu to add cases
4. Add the shortcode `[ekwa_gallery]` to any page or post

== Frequently Asked Questions ==

= How do I add a new case? =

Go to BA Gallery > Add New in your WordPress admin. Enter a title, description, select categories, and add your before/after image sets.

= Can I add multiple before/after images for one case? =

Yes! Each case can have multiple image sets, allowing you to show different angles or stages of the transformation.

= How do I customize the colors? =

Go to BA Gallery > Settings to change the accent color and other appearance options.
